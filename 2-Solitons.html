2-Solitons

2ソリトン共鳴と「計算場」：縮小宇宙で advanced 解が意味を持つ理由

本ノートは、次の主張（証明スケッチ）を、**収束の意味（局所収束）**を明確にした形でまとめる。

物質ソリトン（advanced Hopfion, H=h）と、null 的距離にある放射情報ソリトン（retarded Hopfion, H=+1）からなる 2ソリトン配置が、縮小宇宙（共形時間 η→∞）の下で、少なくとも物質ソリトン中心近傍において H=h+1 の 1物質ソリトン解へ収束する。

結論は二段階：

  • 強い形（局所化データ）：更新後の差場が外向き放射として殻状に局在していれば、中心近傍では 有限の共形時間で完全に一致しうる（Huygens 的描像）。
  • 弱い形（尾がある一般）：尾（tail）が残る場合でも、差場の 局所エネルギーは η→∞ で 0 に落ちる（局所収束）。

これにより、縮小宇宙で advanced 解を「未来側畳み込みを担う計算場」として扱うとき、2ソリトン共鳴（核更新）→外向き放射→近傍は新核へ収束という流れが、Maxwell の線形性・一意性と整合する。

0. 前提：縮小宇宙（共形平坦）と Maxwell の共形不変性

縮小宇宙を共形平坦として

ds2 = a(η)2(−c2 dη2+dx2),   a(η) ↓ 0 (η→∞).

と書く。ここで (c/s) は a(η) の具体形（縮小率）に吸収してよい。

4次元 Maxwell 方程式は（真空・4次元で）共形変換と相性が良く、共形平坦時空では共形座標 (η, x) を用いて「Minkowski 上の Maxwell と同型に扱える」という立て付けが取れる。これが「空間も時間も同じ比率で縮むと Maxwell が連続的に維持される」という主張の数学的な背骨になる。

この結果、以後は共形時間 η を「Maxwell が素直に書ける時間」として用い、

  • 伝播速度（因果構造）は η 上で常に c
  • “null 的距離”も η で素直に表現できる

という環境が整う。

1. 何を「収束」と呼ぶか：中心近傍の局所収束

物質ソリトン中心（収束点）を x = 0 とし、任意の固定半径 ρ > 0 に対して

Kρ := {x ∈ ℝ3 ∣ |x| ≤ ρ}.

上で 2ソリトン場 F^(2) が 1ソリトン場 F^(1)_(h+1) に近づくことを、局所収束として定義する。

収束のノルムとしては、次のいずれかを採用できる：

  • 点wise 収束（強い）
  • 局所エネルギー収束（扱いやすい標準形）

ここでは後者（差場の局所エネルギーが 0）を主に用いる：

δF(η,x) := F(2)(η,x) − Fh + 1(1)(η,x).

Eρ(η) := ∫|x| ≤ ρ(|δE(η,x)|2+|δB(η,x)|2) d3x → 0  (η→∞).

2. 2ソリトン解と 1ソリトン解の書き方

2.1 1物質ソリトン解（H=h と H=h+1）

物質ソリトン（advanced Hopfion）を、共形座標での Maxwell 解として

Fh(1)(η,x),   Fh + 1(1)(η,x)

で表す。H は linking 数（整数）としての内部ラベル。

2.2 2ソリトン解：物質（h）＋情報（+1）

情報ソリトン（retarded Hopfion）を

Finfo(η,x)

とし、初期時刻 η=η0 において「中心から null 的距離」にある（η=η* で中心近傍へ到達する）ように配置する：

|xinfo(η0)| = c(η⋆−η0).

2ソリトン場を形式的に

F(2)(η,x) = Fh(1)(η,x) + Finfo(η,x) + (interaction correction).

と書く。

3. 核心：共鳴（核更新）後に「差場」が外向き放射として分離する

FDB 拡張の更新則（共鳴）では、

  • H: h → h+1 のジャンプが起きる
  • 増分はトポロジカル保護される
  • 増分に対応する retarded 情報ソリトンが放射される（書き込み放射）

とする。

更新が終わった後の時刻を η1 > η* とし、中心近傍の小球

BR := {x ∈ ℝ3 ∣ |x| ≤ R}.

の外では相互作用項が消える（＝核更新は局所）と仮定する。すると差場

δF(η,x) := F(2)(η,x) − Fh + 1(1)(η,x)

は η ≥ η1 で（少なくとも |x| ≥ R の領域で）ソースのない Maxwell を満たす：

d δF = 0,   d  * δF = 0   (η≥η1, |x|≥R).

従って「2ソリトンが 1ソリトンへ収束する」は同値に

δF が中心近傍で 0 に収束する

を示せばよい。

4. η→∞ で δF→0 が言える理由（2ルート）

4.1 ルートA：局所化していれば有限時間で消える（強い）

もし δF(η1,·) が 外向き放射で、ある殻状領域（例：R<|x|<R+Δ）に実効的に局在しているなら、有限伝播速度と（3+1 の波動方程式の）Huygens 的描像により、

  • 波は光円錐（球面）に沿って伝播し
  • 波面が通過した後の内側に尾が残らない（理想化）

という状況が成立する。

結果として、任意の ρ<R に対して

$$ \eta_\rho:=\eta_1+\frac{(R-\rho)+\Delta}{c} $$

以降、中心近傍で

δF(η,x) = 0   (η≥ηρ, |x|≤ρ)

が言える（η→∞ どころか 有限時間で一致）。

これは「情報ソリトンが null に来て共鳴し、以後は書き込み放射として外へ出る」像と、数学的に最も相性が良い“強い”ルート。

4.2 ルートB：尾が残っても局所エネルギーは 0 へ（弱いが一般）

Hopfion のような構造は厳密にはコンパクトサポートでないことが多く、ルートAの「有限時間でゼロ」は一般には期待できない。その場合でも、δF を自由 Maxwell の放射成分と見なせるなら、局所領域に残るエネルギーが時間とともに減る（局所エネルギー減衰）という形で

Eρ(η) → 0   (η→∞)

が期待できる（束縛モードが F^(1)_(h+1) 側に吸収され、δF が放射として外へ出る、というモデル設計と整合）。

この弱い収束でも、中心近傍での「見かけ上の核」は H=h+1 側へ近づく。

5. H-index（linking 数）が h+1 に固定されること

H をリンク数（整数）として定義している以上、「最終的に H=h+1 の 1物質ソリトンになる」には近傍での位相的整合が必要：

H[F(2)(η,⋅)]|Kρ = h + 1   (十分大きい ηで).

ここは共鳴（核更新）の定義そのものとして、

  • 共鳴時刻 η* で情報ソリトン（単位 +1）が核に接続される
  • 以後、核内部はトポロジカル保護により整数値が変化しない（追加の特異過程を許さない）

を要請すれば固定される。

さらに 4章の局所収束（δF→0）があれば、近傍の場は F^(1)_(h+1) と同型になり、近傍の不変量も h+1 に一致する。

6. 「更新直後が外向き放射」になる十分条件（線形性＋一意性）

縮小宇宙（共形時間 η）での真空 Maxwell は、（η を通常時刻と思って）次の2点だけを使えばよい：

  1. 線形性：解の和は解
  2. Cauchy 問題の一意性：同じ初期データからは同じ解しか出ない

更新時刻を η=η*、中心を x=x_c とし、

  • 新しい物質ソリトン核（advanced, H=h+1）を F_m^(h+1)(η,x; η*,x_c)
  • 同一点からの放射情報ソリトン（retarded, H=+1）を F_rad^(1)(η,x; η*,x_c)

と書く。

ここで更新条件を「更新直後の Cauchy データが和に一致する」として

F+(η⋆,x) = Fm(h+1)(η⋆,x) + Frad(1)(η⋆,x).

（同様に E, B の初期データでも一致）

と置く（制約 div E=0, div B=0 も各項が満たせば和も満たす）。

すると線形性により F_sum := F_m^(h+1) + F_rad^(1) は η>η* の真空 Maxwell の解であり、η=η* の初期データが F_+ と一致する。Cauchy 一意性から

F+(η,x) = Fm(h+1)(η,x) + Frad(1)(η,x)   (η≥η⋆).

が従う。よって核との差分は

δF(η,x) := F+(η,x) − Fm(h+1)(η,x) = Frad(1)(η,x).

となり、**非核成分は純粋に retarded（外向き放射）**である。

この十分条件が成立すれば、4章の「外向き放射→近傍収束」へ直結する。

7. まとめ

「縮小宇宙で Maxwell を保ったまま advanced 解が計算場として意味を持つ」ことを、2ソリトン共鳴の観点から整理すると：

  1. 縮小宇宙を共形平坦で表し、共形時間 η で Maxwell を Minkowski 的に扱う。
  2. 共鳴（核更新）後は F^(2) = F^(1)_(h+1) + δF と分解でき、δF は（核外で）自由 Maxwell の放射成分になる。
  3. δF が外向き放射なら、中心近傍では
    • 局所化していれば有限時間で消える（強い）
    • 一般でも局所エネルギーが 0 へ落ちる（弱い）
  4. トポロジカル保護により、近傍の H は h+1 に固定される。
  5. とくに「更新直後の Cauchy データ＝新核＋同一点からの retarded 放射」という条件は、線形性＋一意性で“外向き放射”を十分条件として保証する。
