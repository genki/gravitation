FDB-chi

FDB における heavy mode χ 粒子の詳細解説

ここでは、これまでの議論で出てきた FDB モデルの heavy mode χ 粒子 を、まとめて整理します。
あくまで「FDB という仮想モデルの中での有効粒子」という立場での話です。

1. 位置づけと基本イメージ

1.1 FDB のざっくりした背景

FDB（仮称）は、次のような絵を前提にしています：

  • 宇宙空間には
    超低周波の電磁波（ULE‑EMW） が「海」のように満ちている。
  • 銀河の円盤・バルジのまわりには
    プラズマ密度の急激な変化（勾配）やカットオフ層があり、
    そこで エバネッセント場（しみ出す電磁場） が形成される。
  • ULE‑EMW とエバネッセント場＋プラズマ勾配の組み合わせによって、
    光や星の軌道が曲げられ、それが「暗黒物質的な重力」として見えている。

この「見かけの重力」を担っているのが FDB の本体で、
χ 粒子はその内部自由度として現れる「重い励起モード（quasi‑particle）」
という位置づけになります。

1.2 χ 粒子とは何か（直感）

  • 元の正体：
    ULE‑EMW セクターの中に存在する 重い固有振動モード。
  • 見かけ：
    標準模型の目線から見ると、
    • 電荷 0
    • 色荷 0
    • 質量は WIMP 級（数十〜数百 GeV）
    • ごく弱く標準模型に結合 という意味で、ダークマター風の粒子 として振る舞う。

ただし ΛCDM 的な DM とは違い、

「宇宙全体に均一に存在する新粒子」
ではなく
「銀河のエバネッセント層の殻の中だけで定義される準粒子」

というのが重要なポイントです。

2. 場の描像とパラメータ

2.1 有効場としての χ

FDBの有効理論では、χ はたとえば 実スカラー場 として書けます：

  • 自由項

$$\mathcal{L}_{\text{free}} = \frac{1}{2}(\partial_\mu \chi)(\partial^\mu \chi) - \frac{1}{2}m_\chi^2 \chi^2$$

  • ここでの mχ は
    • ガンマ線スペクトル（10〜100 GeV 帯）に合わせて
    • 典型的には

mχ ∼ 50〜500 GeV

程度の WIMP スケールを想定。

これは「場の内部モードの固有エネルギー」であって、
宇宙論的な暗黒物質質量密度 Ω とは直接は結び付きません。

2.2 「大きさ」（コンプトン波長）

質量 mχ に対するコンプトン波長は

$$\lambda_C = \frac{\hbar}{m_\chi c}$$

なので、たとえば：

  • mχ = 100 GeV → λC ∼ 2 × 10−18 m
  • mχ = 500 GeV → λC ∼ 4 × 10−19 m

銀河スケール（kpc〜pc）から見れば完全に点状で、

天体物理のスケールでは「点粒子」として扱ってよい

というオーダーです。

3. 生成メカニズム：ULE‑EMW → χ へのモード変換

3.1 どこで χ が生まれるか

χ は「どこでも湧く」わけではなく、

  • 銀河円盤・バルジの周辺
  • プラズマ密度 ne の勾配が大きい領域
  • 電磁波のカットオフ層・反射層
  • そこに立つ エバネッセント場

などが重なった エバネッセント層の殻 の内部で生成される、という想定です。

模式的には、χ の生成源 Qχ は

Qχ(r) ∝ |E⃗ULE(r)|2 |∇ne(r)|2

のように、

  • ULE‑EMW の強さ
  • プラズマ密度勾配
  • エバネッセント場

の組み合わせに依存する形をイメージできます。

3.2 「対生成」としての χ・反χ

χ は場としてはスカラーですが、粒子レベルで見ると

  • χ と反χ のペアが
    ULE‑EMW＋エバネッセント場のエネルギーから 対生成 される
  • 生成された χ／反χ は
    • 銀河の重力ポテンシャル
    • FDB 的な「殻のポテンシャル」 によって局所的に閉じ込められる（準粒子として存在）

…というイメージで扱えます。

4. 安定性と対消滅（寿命スケール）

4.1 なぜ「事実上安定」なのか

χ の数密度を nχ、対消滅断面積を ⟨σv⟩ とすると
対消滅による 1 粒子あたりのタイムスケールは

$$\tau_\text{ann} \sim \frac{1}{n_\chi\langle\sigma v\rangle}$$

銀河中心付近のオーダーを入れると：

  • mχ ∼ 100〜500 GeV
  • 質量密度 ρχ を 数十 GeV/cm³、数密度 nχ ∼ 0.1 cm−3 程度と仮置き。
  • ガンマ線強度からみた有効断面積

⟨σv⟩ ∼ 10−26〜25 cm3/s

すると、

τann ∼ 1026 s

  • 宇宙の年齢：約 4×10^{17} s
  • χ の対消滅タイムスケール：約 10^{26} s

→ 宇宙年齢の 10⁸〜10⁹ 倍長い

したがって、

宇宙論的には χ は「ほぼ安定」とみなしてよい
（ごくわずかな分だけ、対消滅でエネルギーが γ にリークしている）

という性質になります。

4.2 γ 線への変換効率

銀河中心の 20 GeV 帯のガンマ線放射強度は

  • 中心近傍で

Jγ ∼ 10−11 ∼ 10−10 W/m2/sr

程度のオーダーと見積もられます。

一方、FDB heavy mode のエネルギー密度が
ρχ ∼ 10−3 J/m3 程度あるとすると、

$$\frac{j_\gamma}{\rho_\chi} \sim 10^{-26}\ \mathrm{s^{-1}}$$

となり、

FDB 側のエネルギーの 10⁻⁸〜10⁻⁹ 分程度が
宇宙年齢スケールでガンマ線になってリークしている

だけで充分説明できる、というオーダーになります。

5. 空間分布と「殻」の役割

5.1 χ が存在できる領域：殻の内側だけ

FDB に自然な仮定として：

χ が安定して準粒子として存在できるのは
エバネッセント場とプラズマ勾配が強い殻の内部だけで、
殻の外に出ると ULE‑EMW の連続モードに溶けてしまう（χ としての識別が失われる）

という描像が考えられます。

  • 殻の内側
    • χ は固有モードとして定義される
    • 数密度 nχ が意味を持つ
  • 殻の外側
    • その固有モードは存在せず、χ は 元の超低周波 EM モードに再変換
    • 「DM 粒子が銀河系外へ飛び出す」イメージではなく
      「χ の身分が消えて場の海に戻る」イメージに近い

5.2 銀河の重力との関係

ここが ΛCDM と決定的に違うところで：

  • ΛCDM DM
    • DM 粒子の質量分布そのものが 重力源の本体。
  • FDB
    • 重力効果の主役は
      ULE‑EMW＋エバネッセント層の場の構造。
    • χ はその内部で現れる 付随的な heavy モード であり、
      重力の本体ではない。

したがって、

  • χ の質量が DM 並みになくても
  • χ の分布が宇宙全体に広がっていなくても

FDB の重力現象自体は成立しうる、というのが特徴です。

6. ガンマ線観測との関係

6.1 銀河中心 20 GeV 付近のガンマ線

FDB heavy mode χ を採用すると、

  • χχ 対消滅

χχ → SM 粒子 → π0 → γγ

  • あるいは有効演算子

$$\frac{\alpha_\text{eff}}{4\Lambda}\,\chi F_{\mu\nu}F^{\mu\nu}$$

  • などを経由して 10〜100 GeV γ を生成できます。

空間分布は、

Iγ(l,b) ∝ ∫ρχ2(r(l,b,s)) ds

という DM 風の J‑factor 形（ρ²）を持つので、

  • 銀河中心を中心とするほぼ球対称なハロー
  • |b|, |l|≲10〜20° くらいで強い

という「GC excess 風」の分布を素直に再現できます。

6.2 矮小銀河で γ が見えない理由（FDB 的には自然）

矮小銀河（dSph）では：

  • 規模が小さい（半径 ~0.1–1 kpc）
  • ガスやプラズマが極端に少ない
  • 構造が単純（バーやスパイラルもない）

→ エバネッセント場＋プラズマ勾配による χ 生成源 Qχ が
元々非常に小さい と期待されます。

結果として：

  • χ の総量も小さい
  • 対消滅 γ も極めて弱い
  • Fermi などの感度では「見えない」のが自然

という解釈になり、
「矮小銀河から DM annihilation γ が見えない」という ΛCDM/WIMP の tension は、
FDB heavy mode χ にとっては問題になりません。

7. MSP 説との関係

銀河中心の GeV γ excess については、

  • MSP（ミリ秒パルサー）集団起源説
  • WIMP/χ 対消滅説

が競っている状況ですが、それぞれ：

  • MSP 説

    • バルジ＋厚いディスク内の多数のパルサー
    • 点源の重ね合わせ → なめらかな拡散に見える
    • 高緯度・大きな角距離までの球対称ハローは説明しにくい

  • FDB heavy mode χ 説

    • 殻の内部で生成された heavy mode χ が、
      DM ハロー風の分布＋対消滅で γ を出す
    • 球対称ハロー成分を自然に持てる
    • MSP 由来の成分と「混在」している可能性もある

という整理になります。

観測的には：

  • スペクトルだけでは両者の区別は難しく
  • 空間分布（球対称 vs X型バルジ）や点源性の有無で
    どこまで切り分けられるか、という問題になります。

8. 時間発展と準定常性

χ の数密度進化は概ね

$$\frac{dn_\chi}{dt} = Q_\text{prod} - \langle\sigma v\rangle n_\chi^2 - \Gamma_\text{leak} n_\chi$$

のように書けます（漏れ：殻外でのモード溶解など）。

銀河形成史と合わせると：

  1. 銀河形成初期（z≳2）

    • ディスク・バルジ形成 → 殻と勾配が立ち上がる
    • χ の生成 Qprod が増え、nχ が成長。

  2. 中間期

  • χ が増えると ⟨σv⟩nχ2 や漏れが効き始める
    • 生成と消失が徐々に釣り合う。

  1. 現在（z~0）

$$\frac{dn_\chi}{dt} \approx 0$$

の準定常状態

  • 回転曲線や γ 線ハローが大きく時間変化していないことと整合的。

この意味で、

χ は増え続けている途中というより、
すでに「生成＝消失＋漏れ」が概ね釣り合った準定常状態にある

と見るのが自然です。

9. まとめ

  • χ 粒子とは？

    • ULE‑EMW セクターの 重い固有振動モード（準粒子）
    • 質量は WIMP 級（50〜500 GeV）
    • 電荷 0・弱結合で、見かけ上「ダークマター風」に振る舞う。

  • どこで生まれ、どこにいるか？

    • 銀河円盤・バルジ周辺の エバネッセント場＋プラズマ勾配が強い殻の内部
    • 殻の外に出ると ULE‑EMW の連続モードに溶けて χ としての身分を失う。

  • 何を説明するのか？

    • 銀河中心の 10〜100 GeV γ を、χχ 対消滅からの放射として説明しうる。
    • ガンマ線強度のオーダーは FDB エネルギー密度からみて十分小さく、
      エネルギー予算的に自然。

  • 宇宙論への影響は？

    • 宇宙全体に均一に存在する粒子ではないので、
      FLRW 宇宙の暗黒物質成分 Ω に直接は寄与しない。
    • 宇宙論スケールでは「重力源はあくまで FDB 本体（ULE‑EMW＋殻）」であり、
      χ は銀河スケールの局所現象。

  • 矮小銀河・MSP などとの整合

    • 矮小銀河ではガス・プラズマ構造が貧弱 → χ 生成が弱く γ も弱い → 非検出は自然。
    • 銀河中心では MSP 集団の寄与と χ heavy mode 対消滅が「両方」あり得る。

こういう形で整理しておくと、
FDB heavy mode χ を「DM 的な見え方をするが、本体は ULE‑EMW の内部励起」という立場で
一貫して扱いやすくなると思います。
