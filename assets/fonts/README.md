# Local font assets

- `NotoSansSymbols2-Regular.ttf` — Google Noto Sans Symbols 2 (Apache License 2.0)
- `NotoSansMath-Regular.ttf` — Google Noto Sans Math (Apache License 2.0)

These fonts are bundled to provide extended glyph coverage (superscripts,
mathematical symbols) for Matplotlib renders used in SOTAとレポート図版.
Source: https://github.com/googlefonts/noto-fonts (commit snapshot mirrors main).
