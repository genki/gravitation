二層モデル（等方＋追加）

等方＝星＋ガス質量: Φ_GR-eq = −G ∫ [ρ★ + gas_scale·ρ_gas]/r d^3y。Υ★のみ銀河固有、gas_scale は共有固定（例 1.33）。

追加＝界面: Hα/X線→ n_e → ω_p → ω_cut から遷移層を抽出し、薄層体積化 S(r) を構成（ε≈H_cut, χ=ℓ*/H_cut, K(θ;χ)）。Φ_FDB は ρ_eff = ρ★+gas_scale·ρ_gas + α_esc·S(r) により FFT/FMM で評価。

μ0(k→∞)=1 により太陽系 Null を保持。等方側に減衰は掛けず、重複を避けます。
