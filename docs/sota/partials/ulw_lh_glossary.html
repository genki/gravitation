用語と定義（ULM‑D / ULM‑P；旧称 ULW‑l / ULW‑h）

  • ULM‑D: プラズマ遮断周波数の上限 ω_cut より低い帯域（視線上の最大プラズマ周波数で定義）。伝搬モードを持たず、局所で吸収・熱化・再放射。遠方 Λ 勾配には直接は寄与しない。
  • ULM‑P: ω ≥ ω_cut の伝搬可能帯域。FDB の担い手。形状に由来する角度因子を保持し、円盤/棒で“中心向き追加”を生む。
  • ULM‑Pa / ULM‑Pb: ULM‑Pa は源から直接の P 成分。ULM‑Pb は ULM‑D が局所プラズマで D→P に再処理された二次 P 成分（等方単極が主）。

脚注: ω_cut は観測線に沿った最大プラズマ周波数で定義し、ULM‑D の総量は等方単極（角平均1）へ吸収（サムルール規格化）。
