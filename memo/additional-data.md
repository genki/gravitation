# 追加で取得を検討するデータセット一覧（FDB関連）

## 銀河スケール
- **THINGS / VLA HI 3D datacube（NGC 2403, 3198 など）**  
  *用途*: 反り・厚み・プラズマ勾配を直接見る。  
  *ステータス*: 未取得。`www2.mpia-hd.mpg.de/THINGS/Data/` への直リンク試行（NGC2403/3198 NA cube, moment0）→ 404。正式ファイル名・パス要確認のうえ再試行。

- **Hα / HII マップ（SINGS / LVL など）**  
  *用途*: 電離ガス分布 → エバネッセント帯の形状把握。  
  *ステータス*: 未取得。

- **CGM 電子密度プロファイル（eROSITA / Chandra / XMM）**  
  *用途*: 銀河外縁のプラズマ殻厚の推定。  
  *ステータス*: 未取得。

## 宇宙網・銀河団
- **宇宙網フィラメントカタログ（SDSS/DESI など）**  
  *用途*: FDB カーネルの大スケール構造、レンズ補正の検証。  
  *ステータス*: Tempel+2014 DR8 フィラメントカタログを取得試行。`dr8_filaments.fits` はサイズ大でタイムアウト、未保存。再試行が必要。

- **銀河団ガス分布（Planck SZ / ACCEPT / Chandra サンプル）**  
  *用途*: M–T 散布の説明に必要な n_e(r)。  
  *ステータス*: 未取得。

## 銀河中心・矮小銀河
- **GC 電子密度マップ（scattering screen, free–free emission）**  
  *用途*: heavy-mode χ 生成強度の推定。  
  *ステータス*: 未取得。

- **dSph 速度分散・3D 密度（Walker+、Gaia DR3）**  
  *用途*: FDB 効果の非検出を定量化。  
  *ステータス*: 未取得。

## 衛星銀河
- **MW/M31 衛星の 6D 速度（Gaia DR3 + HST）**  
  *用途*: VPOS/GPoA の持続性検証。  
  *ステータス*: 未取得。

## 備考
- 大容量データ（特に HI datacube, X-ray FITS）はネットワーク・容量上の制約を確認した上で段階的に取得予定。
- 取得後は `data/` 以下に格納し、`memo/data.md` の目録に反映する。
