# 観測制約の整理(ULW-EM仮説)

- プラズマ遮断: ISMのプラズマ周波数 ν_p ≳ 数百Hz〜kHz。
  - 星系スケール周期(≪Hz)は自由伝搬不可 → 準静的有効場の妥当性。
- 偏波/ファラデー回転: 電界・磁界の上限と比較可能な上限式を作成。
  - 近傍銀河のRM上限, LOFAR/GMRTの偏波測定との整合を試算する。
- 加熱: j_EM由来の有効場が熱入力を与えないモデル化(情報的ポテン
  シャル)であることを定式化。加熱上限はSFR/IRXと矛盾しない範囲。
- PPN/ソーラー系: λが銀河スケール、かつ局所j_EMが弱いとき
  g_ULW≪10⁻¹³ m s⁻² とする抑制条件で制約を回避(数値試算を付録へ)。
- レンズ: 有効ポテンシャル Φ_eff = Φ_GR + ηφ から光路影響を計算。
  - Einstein半径の補正を一次で見積もり、制約に準拠すること。
