# FDB加速度の正しい表式(置換理論として)

更新: 2025-08-27

## 前提

- 場: `(∇^2 - λ^{-2}) φ = β j_EM`, `g = -η ∇φ`。
- 正規化: `A ≡ βη c`(下記で定義する`c`は`j_EM→ρ_eff`変換係数)。

## 3D(体積)の核表現

```
g(r) = A ∫ K_Yuk(Δ; λ) · j_eff(r') dV',  Δ ≡ r - r'
K_Yuk(Δ; λ) = (Δ / 4π |Δ|^3) · (1 + |Δ|/λ) · e^{-|Δ|/λ}
```

- `λ → ∞`で`K_Yuk → K_Newton ≡ Δ/(4π|Δ|^3)`。
- よってFDBは単独でGR同型成分を含む(置換理論)。

分解表示(見通し用):

```
K_Yuk = K_Newton + δK_λ,
δK_λ = K_Newton · ((1 + d/λ) e^{-d/λ} - 1), d=|Δ|
```

較正(例): 有効質量密度`ρ_eff = c · j_EM`が点的に`M`を与える状況で、
`λ→∞`における点源加速度が`GM/r^2`に一致するよう`A c/(4π) = G`を
課す(スケール設定)。

## 2D薄円盤(面密度近似)

```
φ(R) = β ∫ G_2D(d; λ) j_eff(R') d^2R',
G_2D(d; λ) = (1/2π) K0(d/λ),  d = |R - R'|
g(R) = -ηβ ∫ ∇_R G_2D(d; λ) j_eff(R') d^2R'
∇_R G_2D = -(1/2πλ) K1(d/λ) · (R - R')/d
```

`K0`, `K1`は変形ベッセル。`λ→∞`極限でニュートン核に一致。

## 有限λ補正の見通し

有限`λ`では核がユカワ型に変わる。見通しのため

```
K_Yuk = K_Newton + δK_λ,
δK_λ = K_Newton · ((1 + d/λ) e^{-d/λ} - 1), d=|Δ|
```

と分けてもよいが、理論としてはFDB単体で完結する。

## まとめ(最終式)

```
g(r) = A ∫ (Δ / 4π |Δ|^3) (1 + |Δ|/λ) e^{-|Δ|/λ} · j_eff(r') dV'
```

ここで`A`の較正を`λ→∞`でGRと一致するように取れば、FDB単体で重力を
与え、別途`g_GR`を加える必要はない。
