# ULW-EM二乗減衰成分がGR同型加速度となる理由

更新: 2025-08-27

## 要約

FDBは有効場`(∇^2-λ^{-2})φ=β j_eff`, `g=-η∇φ`により決まる。
λ→∞または距離`d≪λ`の極限でポアソン方程式`∇^2φ=β j_0`となり、
グリーン関数は`G_N(d)=(1/4π)(1/d)`、核は`K_Newton=Δ/(4π|Δ|^3)`に収束。
よってFDB内部の二乗減衰成分はGRニュートン極限と同型の加速度を
与える。係数較正`A≡βηc`と`ρ_eff=c j_0`により`λ→∞`でGRと一致する。

## 前提と記号

- 二乗減衰成分: 散乱・吸収が無視でき、強度が`I∝1/r^2`で落ちる
  自由空間伝播成分。
- 有効場方程式: `(∇^2-λ^{-2})φ=β j_EM`, `g=-η∇φ`。
- 実効源: `ρ_eff=c j_0`(単位合わせ用の比例定数`c`)。
- 較正条件: `A≡βηc`を`λ→∞`極限でGRに一致するように取る。

## 1/r^2の起源(幾何・場・伝播子の三視点)

1) 幾何(ガウスの法則): 3次元で点状放射のフラックス保存から、球面
   での流束密度は`1/(4πr^2)`。この幾何学的二乗減衰は、中心力場の
   基本形(逆二乗則)を規定する。

2) 場の方程式(ポアソン極限): `λ→∞`または`d≪λ`で`∇^2φ=β j_0`。その
   基本解は`φ(r)=β∫G_N(|r-r'|)j_0(r')dV'`, `G_N=1/(4πd)`。
   勾配を取れば`g=-η∇φ=A∫K_Newton(Δ)j_0(r')dV'`となる。

3) 伝播子の質量ゼロ極限: `(∇^2-λ^{-2})`はスカラーのユカワ作用素で、
   伝播子は`~e^{-d/λ}/d`。`λ→∞`で質量ゼロの`1/d`伝播子に戻る。
   勾配は逆二乗核に一致する。

## 実空間での同型性の明示

```
φ(r) = β ∫ G_N(|Δ|) j_0(r') dV',  G_N(d) = 1/(4π d)
g(r) = -η ∇φ(r) = A ∫ K_Newton(Δ) j_0(r') dV'
K_Newton(Δ) = Δ/(4π|Δ|^3),  Δ=r-r'
```

GR(ニュートン極限)では`∇^2Φ=4πGρ`かつ`g_GR=-∇Φ`。写像`ρ=ρ_eff=c j_0`と
係数対応`A c/(4π)=G`を課すと、`g`と`g_GR`は形とスケールが一致する。

## フーリエ空間での同型性

`∇^2φ=β j_0`のフーリエ像は`-k^2 φ(k)=β j_0(k)`、よって
`φ(k)=-β j_0(k)/k^2`。加速度は`g(k)=i k η φ(k)`より
`g(k)= -i k (βη) j_0(k)/k^2`。これは`k`方向の単位ベクトルに沿う
`1/k`型の応答で、逆変換により実空間の逆二乗核に一致する。
GRニュートン極限でも`Φ(k)=-4πG ρ(k)/k^2`で同型の構造となる。

## どのような条件で成り立つか

- 時間変化: ULW-EMは準静的(源・媒質の時間スケールが伝播時間より
  長い)。遅延・放射項は無視できる。
- 媒質: 散乱・吸収が無視できる。`j_EM≈j_0`。
- 線形・等方: 源と場の結合が線形、等方的。異方性が強い場合は核が
  歪み得る。
- スケール: 対象距離`d`が`λ`より十分小さい、または`λ→∞`。

## 有限λ補正の関係

FDB核`K_Yuk=K_Newton·(1+d/λ)e^{-d/λ}`は、
`K_Newton`(GR同型)と補正`δK_λ`に分解できる。

```
K_Yuk = K_Newton + δK_λ,
δK_λ = K_Newton · ((1 + d/λ) e^{-d/λ} - 1)
```

## 実装への写像と較正

`src/models/fdbl.py`のYukawaソルバーにおいて、`lam_kpc`を十分大きく
設定し、`j_eff=j_EM`とすれば、数値的にGR同型の`g`が得られる。
`A=βηc`を`λ→∞`で`G`に一致するよう較正する。有限`λ`ではユカワ補正が
上乗せされる。

## まとめ

ULW-EMの二乗減衰成分は、(i)ガウスの法則、(ii)ポアソン極限、
(iii)質量ゼロ伝播子の三視点のいずれからも逆二乗核を帰結する。
よってFDB理論の内部に、GRニュートン極限と同型の加速度成分が自然
に現れる。残差は有限`λ`に由来する補正が担う。
