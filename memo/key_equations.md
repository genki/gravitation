# 主要方程式メモ

更新: 2025-08-27

本リポジトリのFDB置換理論で最も重要な方程式は、
ユカワ型ポアソン方程式と、その勾配で定義される加速度です。

- フィールド方程式:
  `(∇^2 − λ^{-2}) φ(r) = β j_EM(r)`
- 加速度定義:
  `g(r) = −η ∇φ(r)`

ここで、`φ`は情報バイアス場、`j_EM`はEM放射の代理量、
`λ`は相関長、`β, η`は結合・換算係数です。`λ→∞`でニュートン核
に収束し、GRのニュートン極限を再現します。
