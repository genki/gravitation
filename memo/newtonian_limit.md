# GRのニュートン極限メモ

更新: 2025-08-27

## 定義
- 一般相対論の方程式と運動方程式を、弱重力・低速・緩変化の
  近似で漸近展開し、ニュートン重力理論(ポアソン方程式と
  質点運動)に還元する極限。形式的には `c→∞`、または
  `ε→0` (`ε∼v/c∼√(|Φ|/c^2)`) の極限。

## 前提(オーダリング)
- ポテンシャル比: `|Φ|/c^2 ≪ 1` (弱重力)
- 速度: `v/c ≪ 1` (低速)
- 圧力: `P ≪ ρ c^2` (静水圧はエネルギー密度に比して小)
- 時間変化: `∂_t g_{μν}` は空間変化より高次 `O(ε^3)`
- ソース: `T_{00}≈ρ c^2`, `T_{0i}≈ρ c v_i`, `T_{ij}≈P δ_{ij}`

## 計量の展開(代表形)
- `g_{00} = −(1 + 2 Φ/c^2) + O(ε^4)`
- `g_{ij} = (1 − 2 Φ/c^2) δ_{ij} + O(ε^4)`
- `g_{0i} = O(ε^3)` (定常近似で無視可能)

## 方程式と運動への帰結
- アインシュタイン方程式 → ポアソン方程式:
  `∇^2 Φ = 4 π G ρ + O(ε^4)`
- 測地線方程式 → ニュートン運動:
  `d^2 x/dt^2 = − ∇ Φ + O(ε^2)`

## 本リポジトリの理論との関係
- `λ→∞` でユカワ核がニュートン核に一致し、
  `g = −η ∇φ` がGRのニュートン極限に整合:
  `(∇^2 − λ^{−2}) φ = β j_EM  →  ∇^2 φ = β j_EM`。
