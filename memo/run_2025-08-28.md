# 近傍銀河・外縁重みテスト(実行ログ要約)

- 実行日: 2025-08-28
- 目的: 既存nearbyセットに対し外縁重みを強化し(`--outer-weight 1.5`)、ULW(＋1/r)の感度を軽量に確認。
- 実行:
  - `PYTHONPATH=. .venv/bin/python scripts/compare_fit_multi.py \
     --names-file data/sparc/sets/nearby.txt --boost 0.5 --boost-tie-lam \
     --auto-geo --gas-scale-grid 1.0,1.33 --outer-weight 1.5 \
     --out data/results/multi_fit_nearby_outer1.5.json`
  - `PYTHONPATH=. .venv/bin/python scripts/build_report.py`

## 結果サマリ(nearby, outer-weight=1.5)
- 対象: `DDO154, NGC2403, NGC3198, NGC6946, NGC2903, DDO161, D631-7, DDO064`
- ULW最良: `λ≈15 kpc, A≈0.01, gas_scale=1.33`
- 合計χ²(ULW): ≈ 5.71e3  (ファイル: `data/results/multi_fit_nearby_outer1.5.json`)
- レポート: `server/public/reports/multi_fit_nearby_outer1.5.html`

### 参考(前日nearbyの基準)
- 2025-08-27のnearby: `λ≈5 kpc, A≈31.6, gas_scale=1.33`, 合計χ² ≈ 1.83e5
- 注意: 外縁重みや誤差下限の設定が異なるため、総χ²の絶対比較は注意が必要（重みスキーム変更によりスケールが変化）。指標は相対差・曲線形状の傾向確認に利用。

## 所感・観察
- 外縁重みを強めると、1/r成分(`alpha_line`)が各銀河で大きめに最適化され、`λ`は長め(≈15 kpc)へシフト。外縁の傾き合わせを優先した結果と整合。
- A(1/r²)が極小(0.01)へ寄る解が選好された。重み付けの強化で1/r主導のフィットに寄りやすい可能性。過剰適合を避けるため、`inv1-orth`や`eg-frac-floor`の再適用で安定性を確認したい。

## 次アクション案(提案)
