# 拡大20: 外縁重みと正則化長εの感度比較 (軽量解析)

- 実行日: 2025-08-31
- 目的: 既存結果(JSON)を横断比較し、共有パラメータ(λ, A, gas_scale)固定下で
  外縁重み(`--outer-weight`)と1/r正則化長εの影響を定量確認。
- 背景: 2025-08-29までの実行で、直交化(`--inv1-orth`)と誤差下限(`--eg-frac-floor=0.15`)
  導入により安定化を確認。残る感度(外縁重み, ε)を整理して次の一手を決める。

## 結果サマリ
- 共有パラメータは全ケースで共通: `λ=15 kpc, A=100, gas_scale=1.33`。
- 合計χ²(ULW)とΔAIC(=AIC_ULW−AIC_GR):
  - 基準: χ²=911.02, ΔAIC=-3540.15
  - 外縁1.5: χ²=930.16, ΔAIC=-3521.01 → わずかに悪化
  - ε=0.3: χ²=917.21, ΔAIC=-3533.96
  - ε=0.5: χ²=911.02, ΔAIC=-3540.15 (基準同等)
  - ε=0.8: χ²=900.90, ΔAIC=-3550.27 → 最良(わずかに改善)
- 解釈: 直交化+誤差下限の下では、
  - 外縁重みをさらに強める(1.5)メリットは限定的(全体では軽い悪化)。
  - 1/r正則化長はε≈0.8kpcが最も良好(差は小さいが一貫)。
