# LSB/HSBのalpha_line効果量とCIの算出（補足解析）

- 実行日: 2025-09-01
- 目的: 既報の有意性検定（KS/MWU）に加え、群間差の大きさを
  効果量（Cliff's δ, Cohen's d, Hodges–Lehmann推定）で定量し、
  中央差の信頼区間をブートストラップで与える。

## 実行
- 入力: `assets/results/alpha_line_lsb.csv(59)`, `alpha_line_hsb.csv(59)`
- スクリプト: `scripts/analyze_alpha_line_effects.py`（依存は標準+matplotlibのみ）
- 出力: `assets/results/alpha_line_effects.json`, 図`assets/figures/alpha_line_violin.png`

## 結果サマリ
- Hodges–Lehmann(HSB−LSB): 約 +4.35（代表的な位置差）。
- 中央差の95% CI(パーセンタイルB=2000): [約 +2.11, +9.55]。
- Cohen's d: 約 +0.36（小〜中の効果）。
- Cliff's δ(LSB vs HSB): 約 −0.34（HSBが大きい方向。|δ|は中程度）。
- 解釈: 先行の有意差（p<0.01）に整合。効果量は「小〜中」規模で、
  実質的にも無視できない差が示唆される。
