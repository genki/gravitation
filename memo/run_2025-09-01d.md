# 図面のΔV(残差)パネルの可読性改善

- 実行日: 2025-09-01
- 目的: 代表図の残差(ΔV)パネルが低すぎて見づらい問題の改善。
  既定高さを拡大し、CLIから柔軟に調整可能にする。

## 実行
- 対象: `scripts/compare_fit_sparc.py` のSVG描画(`svg_overlay`)。
- 変更点:
  - 既定の残差パネル高さを 120→160 px に拡大。
  - 新オプション `--res-height <px>` を追加（任意の高さに変更可能）。
  - コード: `svg_overlay(..., res_height=int)` を導入し `--res-height` を連携。
- 検証: DDO154 で `--res-height 200` を指定して再生成。

## 結果サマリ
- ΔVパネルの視認性が向上（目盛とゼロ線の余白確保、トレース判別性改善）。
- 既定値のままでも従来比 +33%の高さ（120→160px）。
- バッチ生成系は追加指定なしで恩恵（既定拡大）。

## 生成物
- 修正: `scripts/compare_fit_sparc.py`
- サンプル図: `paper/figures/compare_fit_DDO154_upd.svg`（`--res-height 200`）
