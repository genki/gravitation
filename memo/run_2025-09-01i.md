# 銀河プロファイル一覧ページの作成

- 実行日: 2025-09-01
- 目的: 自動生成した銀河ごとの調査メモ（Wikipedia要約＋フラグ）を
  一覧で横断できるページを追加し、ブラックリスト/除外状況の確認と
  フィルタを容易にする。

## 実行
- 追加: `scripts/build_galaxy_profiles_page.py`
  - 入力: `memo/galaxies/index.json`, `data/sparc/sets/blacklist.txt`, `exclude_env.txt`
  - 出力: `server/public/galaxies/index.html`（検索ボックス付きテーブル）
  - 列: Galaxy / Flags / Status(blacklist/excluded/ok) / Memoリンク
- サイトへ組込み: `scripts/update_and_build_site.sh` に呼び出しを追加（2.5/3ステップ）。

## 結果サマリ
- `https://localhost:3131/galaxies/` で一覧ページを参照可能。
- ブラックリストや手動除外のステータスが一目で確認でき、
  検索ボックスでフィルタ可能。

## 生成物
- スクリプト: `scripts/build_galaxy_profiles_page.py`
- HTML: `server/public/galaxies/index.html`
