# 共有(λ,A,gas)フィット（ブラックリスト考慮）

- 実行日: 2025-09-02
- 目的: 外的要因が強い対象（ブラックリスト）を除外し、共有パラメータの
  フィットを実施。従来の全件(175)から除外後のクリーン母集団で再評価する。

## 実行
- 入力セット: `data/sparc/sets/clean_for_fit.txt`（なければ`all.txt`）
- 除外: `data/sparc/sets/blacklist.txt` と `exclude_env.txt`
- 生成セット: `data/sparc/sets/clean_no_blacklist.txt`（kept=100）
- コマンド:
  ```sh
  PYTHONPATH=. .venv/bin/python scripts/run_shared_fit_filtered.py
  ```
  - オプション（内部）: `--lam-grid 15,18,20,24`, `--A-grid 80,100,125`,
    `--gas-scale-grid 1.0,1.33`, `--boost 0.5 --boost-tie-lam --auto-geo`
    `--pad-factor 2 --eg-frac-floor 0.15 --inv1-orth --line-eps 0.8`

## 結果サマリ
- サンプル: 100銀河（BL 9件, manual 0件除外）
- 最良: `λ≈20 kpc, A≈125, gas_scale≈1.33`
- 合計χ²: GR≈2.02e4 → ULW≈4.96e3（改善倍率×4.08）
- 所見: BL除外後も共有(λ=20, A=125, gas=1.33)が安定。外乱の強い対象を
  省くことで総χ²が相対的に低下し、代表的傾向の見通しが明瞭。

## 生成物
- JSON: `data/results/multi_fit_all_noBL.json`
- HTML: `server/public/reports/multi_fit_all_noBL.html`
- セット: `data/sparc/sets/clean_no_blacklist.txt`
