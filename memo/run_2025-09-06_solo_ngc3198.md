# 単一銀河ベンチ（NGC 3198）— 整合・統計の是正とページ更新

## 結果サマリ
- MONDの単位不整合を修正（a0=1.2e-10 m/s^2 → (km^2 s^-2)/kpc 換算で約3702.8）し、GRと同値になる不具合を解消。
- GR+DM(NFW)において各グリッド点でμ(M/L)を再推定するよう改善。GRより大幅に悪化する症状は解消し、χ²はGRと同程度に収束。
- ベンチページをAIC→AICc表記へ統一し、(N,k)を併記。ELPDは従来通り併載。
- 薄層幅εは固定ポリシー（既定0.8 kpc、H_cutがあればそれを採用）に変更。診断用に0.5/1/2倍のスイープは残すが最終報告は固定εの統計を採用。
- β（Lambert→前方化）は未配線で効果ゼロな実装を明記し、ベンチではβ=0（OFF; 自由度に数えない）とする注記を追記。
- 外縁1/r²復帰の簡易可視化CSVを追加（g(R)·R²）。
- 太陽系Null（μ0(k→∞)→1）のログを自動添付（存在時）。
