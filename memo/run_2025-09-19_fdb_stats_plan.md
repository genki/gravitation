# 2025-09-19 FDB固有予測統計 拡張計画

## 目的
- 幾何“陰影”、外縁1/r²復帰、遮蔽非対称の 3 指標のうち 2/3 で p<0.01 を達成するため、多天体統計フローを確立する。

## 現状
- `analysis/shadow_bandpass.py` と `scripts/galaxies/compute_fdb_signatures.py` がバレット向けに整備済み。
- 単一銀河ベンチ UI（外縁1/r²、Hα 等高線、ω_cut 誤差伝播）を追加銀河へ展開可能。

## 施策案
1. **テンプレート化**: ベンチ HTML の指標カードを Jinja テンプレ化し、対象銀河ごとに自動生成できるよう調整。
2. **一括計測**: `compute_fdb_signatures.py` に CSV/YAML 出力と複数銀河バッチ処理オプションを追加。
3. **FDR 管理**: Benjamini-Hochberg の q 値を計算し、ダッシュボードで色分け表示する。
4. **可視化**: 幾何指標の分布、p値ヒストグラム、FDR 閾値をまとめた診断図を `server/public/state_of_the_art/figs/` に追加。

## タスク分担（暫定）
- 実験チーム: コード実装、データ処理、CI 整備。
- 理論チーム: 指標定義レビュー、統計的意味付け、報告文章の監修。

## 次アクション
1. 対象銀河リスト（既存 2 銀河＋候補）を整理し、必要データ（Hα,H I,ω_cut）をチェック。
