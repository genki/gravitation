# ULW‑EM「未来確定バイアス」による**見かけの引力** — 更新版仕様（更新指示付き）

> **要旨**  
> 実在の外力・仕事なしに、**将来の“記録（相互作用）”の成立**に**条件づけられた経路の偏り**が、観測上**引力と等価**な振る舞いを生む。  
> これは確率過程の **Doob の *h* 変換**（条件付き到達確率による測度再重み付け）として**厳密に定式化**できる。  
> 見かけの力は **情報ポテンシャル** \(U_{\text{info}}=\kappa \ln h\) の勾配として現れ、**運動エネルギーの増分は \(U_{\text{info}}\) の減少で釣り合う**（外力なし、仕事なし）。

---

- **[置換]** 旧「ULW の放出・吸収が機械的に粒子を押す」

  *実務*：理論書の「力の起源」を**測度の再重み付け**へ書き換え。

## 1. 公準（Axioms）

1. **対象**：超低周波 EM（ULW‑EM）により、将来の「記録（相互作用）」成立事象 \(E\) が定義できる。  
2. **見かけの力**：力学は変えない。**経路の条件づけ**が**見かけの引力**を作る。  
3. **情報ポテンシャル**：  
   \[
   U_{\text{info}}(x,t)=\kappa\,\ln h(x,t),\qquad
   h(x,t)=\mathbb{P}(E\mid x_t=x)
   \]
4. **保存則（見かけ）**：典型経路に沿って  
   \[
   \frac{d}{dt}\bigl[K(x,\dot x)+U_{\text{info}}(x,t)\bigr]=0
   \]
   （外力・仕事なし。近接以外の可能性の消滅が \(U_{\text{info}}\) 減少として現れる。）

---

## 2. 定式化（Doob の *h* 変換）

- 基準過程（外力なし）：生成子 \(\mathcal{L}\)。将来の記録事象 \(E\) に条件づけた過程の生成子：
  \[
  \boxed{\ \mathcal{L}^{(E)} f = h^{-1}\,\mathcal{L}(h f)\ }
  \]
- 拡散型での**実効ドリフト**（例：ブラウン＋慣性の簡約）：
  \[
  b_{\rm eff}=b+2D\,\nabla \ln h
  \]
- 慣性系の極限（ラグランジアンでの見かけの力）：
  \[
  m\,\ddot x \;=\; -\nabla U_{\text{info}}(x,t) \;=\; -\kappa\,\nabla \ln h(x,t)
  \]

> **解釈**：\(\nabla\ln h\) が**未来確定バイアス**の“方向”を与え、**近接方向**に向く。

---

## 3. 規格化（弱場の整合）

- **質量**：\(\kappa = m c^2,\ \ h=\exp(\Phi_{\rm eff}/c^2)\) と取れば  
  \[
  U_{\text{info}}=m\,\Phi_{\rm eff},\quad \ddot x=-\nabla \Phi_{\rm eff}
  \]
  **等価原理**（加速度の \(m\) 消失）を満たす。

- **光線**（フェルマー原理の見かけ版）：  
  \(U_{\text{info}}^{(\gamma)}=(p/c)\,\Phi_{\rm eff}\) とおくと、偏向角
  \[
  \boxed{\ \hat\alpha \;=\; 2 \int \nabla_\perp\!\bigl(\Phi_{\rm eff}/c^2\bigr)\,ds\ }
  \]
  すなわち **点質量で \(\hat\alpha = 4GM/(b c^2)\)**（弱場の係数2）を再現。

> **注意**：「\(\Phi_{\rm eff}\) は“実在の重力ポテンシャル”ではない**が**、見かけのダイナミクスは**同型**。**

## 4. 幾何バイアスの具体例

### 4.1 直観
観測者に近い点ほど「記録されやすい」→ \(h\) が大 → \(\ln h\) 勾配が観測者方向 → 見かけの引力が**正符号**。

### 4.2 一般式
遠方近似では、球面上の点 \(x\) から観測者までの距離 \(d(x)\) に対し  
\(h(x)\propto 1/d(x)^2\)（立体角・減衰）。従って \(\nabla\ln h = -2\,\nabla\ln d\) が観測者方向を指す。

### 4.3 球殻モデル（半径 \(r\)、観測者距離 \(R>r\)）
球面一様重ね合わせからの放射・検出に**条件づけ**た発信源の事後平均：
\[
\langle z\rangle_{\rm click}
= r\,
\frac{\displaystyle \frac{R^2+r^2}{2Rr}\ln\!\frac{R+r}{R-r}-1}{\displaystyle \ln\!\frac{R+r}{R-r}}
= \frac{2}{3}\frac{r^2}{R} + O\!\left(\frac{r^4}{R^3}\right) > 0
\]
→ **観測者側へ偏る**ことが厳密に確認できる。

## 5. \(\Phi_{\rm eff}\) の設計（モデル自由度）

- **最小**：\(\Phi_{\rm eff}=\Phi_{\rm N}[\rho_{\rm bar}]\)（バリオンのニュートンポテンシャル）
- **拡張A（線形核）**：フーリエ空間で \(\Phi_{\rm eff}(k) = \Phi_{\rm N}(k)\,\mu(k)\)（Yukawa 和など）  
  → \(V(R)\) は Hankel で計算可。
- **拡張B（非線形）**：\( a = \nu(a_N/a_0)\,a_N\) 型（MOND 風の汎化）  
  → RAR/BTFR を**見かけ**で再現するための実用形。
- **要件**：ソーラー系（弱場級数）、等価原理、レンズ（係数2）と**同時整合**。

> **実務**：まずは **最小**→**A**→**B** の順で拘束を強めつつ適合度を比較。

## 6. 観測・物理の整合（チェックリスト）

- **仕事なし**：\(\frac{d}{dt}(K+U_{\text{info}})=0\)（見かけの保存）。  
- **無信号**：\(h(x,t)\) は**現時点の配置**から定義（将来の自由意志での操作は条件づけ後の**事後**）。  
- **等価原理**：規格化により自動達成。  
- **重力レンズ**：\(\hat\alpha\) のアクロマティシティ・係数2。  
- **副作用（像のボケ等）**：デコヒーレンス（散逸）は**副次**、観測上限内に抑制。

## 7. 実装指針（研究実務）

- **情報ポテンシャル層**：  
  - `U_info = m * Phi_eff`（質量）  
  - `alpha = 2 ∫ ∇_⊥ (Phi_eff/c^2) ds`（光）
- **\(\Phi_{\rm eff}\) 層**：  
  - **ニュートン極限**：`mu(k)=1`

## 8. よくある誤解と回答

- **Q**：「Δt を極小にすると効果は消える？」  
  **A**：いいえ。**生成子レベル**（連続時間）で \(\nabla\ln h\) が残るため、**Δt 非依存**。

- **Q**：「外力がないのにエネルギーは？」  
  **A**：観測上は \(K\) 増が **\(U_{\text{info}}\) 減**で**帳尻**。実在の仕事はしていない。

- **Q**：「測定（デコヒーレンス）だけで平均が曲がる？」  
  **A**：**無条件平均**では曲がらない。**条件づけ**が経路を傾け、**見かけの力**になる。
