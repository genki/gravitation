# 理論ノート: ULW‑EM「未来確定バイアス」による見かけの引力 — 更新版

更新: 2025-08-31

## 公準と基本定義
- 見かけの力は**力学を変えず**、将来の記録事象 `E` への**条件づけ**で経路測度が傾く（Doob の *h* 変換）。
- **情報ポテンシャル**: `U_info(x,t) = κ ln h(x,t)`, `h = P(E | x_t = x)`。
- 典型経路での見かけ保存: `d/dt [K + U_info] = 0`（外力・仕事なし）。

## 生成子と見かけドリフト
- 条件づけ過程: `L^(E) f = h^{-1} L(h f)`。
- 拡散型の実効ドリフト: `b_eff = b + 2D ∇ ln h`。慣性極限で `m x¨ = −∇U_info = −κ ∇ ln h`。

## 規格化（弱場）
- 質点: `κ = m c^2`, `h = exp(Φ_eff / c^2)` → `U_info = m Φ_eff`, `x¨ = −∇Φ_eff`。
- 光: `U_info^(γ) = (p/c) Φ_eff` → 偏向角 `α̂ = 2 ∫ ∇_⊥(Φ_eff/c^2) ds`（弱レンズ係数2）。

## 幾何バイアス
- 遠方近似で `h ∝ 1/d^2` → `∇ ln h = −2 ∇ ln d` が観測者方向を指す。
- 球殻モデルの事後平均偏り: `⟨z⟩_click = (2/3) (r^2/R) + O(r^4/R^3) > 0`。

## 実装レイヤと受け入れテスト
- 情報ポテンシャル層: `U_info = m Φ_eff`, `U_info^(γ) = (p/c) Φ_eff`。
- Φ_eff 層: 最小はニュートン核、線形核 `μ(k)`/非線形 `ν` は差し替え可能。
- 受け入れテスト: (1) 点質量 `a = −∇Φ`、(2) 光の偏向 `4GM/(bc^2)`、(3) 薄円盤 `V^2(R)` スモーク。

## 誤解と回答（抜粋）
- Δt→0 で効果は消えない → 生成子レベルで `∇ ln h` が残る。
- 外力なしのエネルギー帳尻 → `K` 増分は `U_info` 減少で補償。
- 測定だけで平均が曲がる？ → 無条件平均では曲がらず、**条件づけ**が偏向を生む。
