What's new (2025‑Q3)

30秒要約 — 「用語を ULM に統一（旧称 ULM）。帯域は ULM‑P/D（旧 h/l）。宇宙では P だけが遠くまで届いて FDB を担う。D は近場で P に化けて“総量（等方）”を底上げする。球は GR と同じ、薄円盤や棒では“中心向き追加”が出て回転曲線の外縁が自然に平坦化する。」

共有ハイパー（採択）：ε=1, k0=0.2 kpc⁻¹, m=2, gas_scale=1.33

補遺: 界面放射（Σ）モデルを追加実装（等値面上のJ_outからΛを評価）。AICc/ELPD等価性を検証中。

方針:

  • FDB（Future Decoherence Bias）を“重力”ではなく見かけの引力として扱う。
  • 太陽系スケールでは GR と同型、銀河スケールでは超長波の自由度で差が出る可能性を検証する。
  • 比較は“観測に対して実運用で使われるベースライン”を含め公正に行う。
  • GR+DM を含める理由: DM は何でも可ではなく、ハロー形状や自由度は有限で、c–M 関係やコア/カスプ等の宇宙論的整合で拘束される。したがって GR+DM は常に一致するわけではなく、フェアなベースラインである。

用語と定義（ULM‑D / ULM‑P；旧称 ULM‑l / ULM‑h）

  • ULM‑D: プラズマ遮断周波数の上限 ω_cut より低い帯域（視線上の最大プラズマ周波数で定義）。伝搬モードを持たず、局所で吸収・熱化・再放射。遠方 Λ 勾配には直接は寄与しない。
  • ULM‑P: ω ≥ ω_cut の伝搬可能帯域。FDB の担い手。形状に由来する角度因子を保持し、円盤/棒で“中心向き追加”を生む。
  • ULM‑Pa / ULM‑Pb: ULM‑Pa は源から直接の P 成分。ULM‑Pb は ULM‑D が局所プラズマで D→P に再処理された二次 P 成分（等方単極が主）。

脚注: ω_cut は観測線に沿った最大プラズマ周波数で定義し、ULM‑D の総量は等方単極（角平均1）へ吸収（サムルール規格化）。

二層モデル（等方＋追加）

等方＝星＋ガス質量: Φ_GR-eq = −G ∫ [ρ★ + gas_scale·ρ_gas]/r d^3y。Υ★のみ銀河固有、gas_scale は共有固定（例 1.33）。

追加＝界面: Hα/X線→ n_e → ω_p → ω_cut から遷移層を抽出し、薄層体積化 S(r) を構成（ε≈H_cut, χ=ℓ*/H_cut, K(θ;χ)）。Φ_FDB は ρ_eff = ρ★+gas_scale·ρ_gas + α_esc·S(r) により FFT/FMM で評価。

μ0(k→∞)=1 により太陽系 Null を保持。等方側に減衰は掛けず、重複を避けます。

現在の加速度の式と一貫性

総加速度 (半径Rの円筒平均近似):

$g_{\mathrm{tot}}(R)=g_{\mathrm{gas}}(R)+\mu\,g_{\mathrm{star}}(R)+g_{\mathrm{FDB}}(R;\,\mu(k))$

FDB の定義（フーリエ空間倍率）:

$\mu(k)=1+\frac{\varepsilon}{1+(k/k_0)^m}$（$\varepsilon>0,\;m\gtrsim2$ は共有。高 $k$ で $\mu\to1$ → 太陽系でGRと同型）

ソースの整合:

$\rho_b=(M/L)\times\text{光度}$ を採用し、$j_{\mathrm{EM}}$ ではなく $\rho_b$ を用いる（式は $\rho_b$ に統一）。

事前: ガス倍率は狭い正規事前（例: 1.33±0.1）。ゲイン係数の次元は μ(k) に吸収し、一元化。

補足: DoG/円盤/棒などの“物理項の重み”は使用しない。必要な強調は損失の半径重み w(R) で行う。

記号の意味（要約）

  • $R$: 半径[kpc]（円筒平均の自変数）
  • $r$: 2D平面での距離（$r=\lvert \mathbf{x}-\mathbf{x}'\rvert$）
  • $\mathbf{x}=(x,y)$: 2D座標
  • $g_{\mathrm{tot}}$: 総加速度，$g_{\mathrm{gas}}$, $g_{\mathrm{star}}$, $g_{\mathrm{FDB}}$
  • $\mu$: 星の有効M/L
  • $\mu(k)$: FDBの倍率関数，$k$: 波数，$k_0,\,m,\,\varepsilon$: 共有定数
  • $\rho_b$: バリオン密度（$\rho_b=(M/L)\times$光度）

記号の意味

  • ε: μ(k)の強度（無次元）
  • k0: μ(k)の特性スケール [kpc⁻¹]（λ≈1/k0）
  • m: μ(k)の指数（無次元）
  • gas_scale: ガス質量スケールの比率（無次元）
注: reduced χ² は rχ² = χ²/(N−k)。N は共通マスクで揃えた有効データ点、k は当該集計に含めた自由度。fold 合算では各 fold の N と k を加算して算出します。共有 μ(k) サマリは kglobal=4（ε,k0,m,gas_scale）。per‑galaxy 表示は銀河内パラメータのみを数え、共有ハイパーは含めません。観測速度誤差には 5 km/s の下限（点毎に clip(0.03×Vobs, 3..7)）を適用。NFW の濃度には c–M 事前（ln c のガウス, σ≈0.35）を加味します。
